# LifeOS 手机端快速录入 v1

当前已提供最小可用交互（MainActivity）：
- Refresh Dashboard
- Quick Add Work 1h
- Quick Add Learning 1h
- Quick Add Income 10000 cents
- Quick Add Expense 3000 cents
- Create Manual Backup

页面下方展示最近记录列表（跨 time/income/expense/learning 聚合排序）。

用途：
- 验证“录入 -> 入库 -> 指标 -> 展示 -> 备份”全链路。
- 在正式 UI 设计前，先满足每日真实可用。
